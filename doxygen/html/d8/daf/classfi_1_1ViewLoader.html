Detailed Description

template<typename UserType>
class fi::ViewLoader< UserType >

View loader task. Take a HTGSViewRequestData and produce a HTGSTileRequestData.

First task of the Fast Image graph. The goal of the ViewLoader is to use the fi::HTGSViewRequestData<UserType> to generate n fi::HTGSTileRequestData<UserType> for the ATileLoader used. The fi::HTGSViewRequestData<UserType> describes which view is requested by the end user. The fi::HTGSTileRequestData<UserType> describes which part of the image has to be loaded and where it has to be copied in the view. It will use the MemoryManager to get an empty view, or wait until a view has been released. The number of views available to the memory manager can be specified from fi::FastImage->getFastImageOptions()->setNumberOfViewParallel().
