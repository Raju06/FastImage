Detailed Description

template<typename UserType>
class fi::View< UserType >

View used by FastImage to represent image's region.

View data class, contains the tiles loaded by the Fast-Image graph. The view is defined by a central tile, and the pixels surrounding it. The views will be pre-allocated by the memory manager. The pixels are located by their local coordinate where the position (0/0) is in the upper left of the central tile.

For example to get once all the pixel of the image, the radius is set to 0, and it can be used:

for (int32_t r = 0; r < view->getTileHeight(); ++r) {
for (int32_t c = 0; c < view->getTileWidth(); ++c) {
// Get the pixel from it's coordinate
pixValue = view->getPixel(r, c);
if (!std::isnan(pixValue)) {
...
}
}
}
