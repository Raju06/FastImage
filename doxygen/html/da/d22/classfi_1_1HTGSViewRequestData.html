Detailed Description

template<typename UserType>
class fi::HTGSViewRequestData< UserType >

Data representing a view request.

FastImage will use this object to request a view. It will be sent to a ViewLoader. The view Loader will then use it to generate HTGSTileRequestData.

Template Parameters
UserTypePixel Type asked by the end user
