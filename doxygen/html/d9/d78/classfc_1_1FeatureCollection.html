Detailed Description

Representation of a mask.

FastImage way to represent and use a mask. Instead to have a pixel centric approach as the mask, it has been chosen to use a Feature centric approach. A Feature is a set of connected pixel representing a separated region of pixels.

An AABB tree is used to provide a quick lookup from pixel to feature.

Convenient methods have been made to create a Feature collection from an image, and the inverse operation.
