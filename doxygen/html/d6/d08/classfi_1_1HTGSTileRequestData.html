Detailed Description

template<typename UserType>
class fi::HTGSTileRequestData< UserType >

Data representing a tile request.

The Tile Request Data represent a specific tile request which will fill the future view. It has been generated by The ViewLoader and sent to the ATileLoader. The ATileLoader will use it to know which tile to request to the cache / file and then copy part or the whole tile into the view. All the positions for the copy (source/destination) are stored into the HTGSTileRequestData.
