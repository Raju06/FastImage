Detailed Description

template<typename UserType>
class fi::FigCache< UserType >

Fast Image cache.

Fast Image cache, which is used to limit the image IO. Is uses a LRU policy. The amount of cached tiles is set up at construction. The cached tiles are saved into a matrix which has the same dimension as the image.
